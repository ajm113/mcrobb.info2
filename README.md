## Installation

**1. Clone this repo:**

```sh
git clone git@github.com:ajm113/mcrobb.info2.git mcrobb.info
cd mcrobb.info
```


**2. Install the dependencies:**

```sh
npm install
```

> You're done installing! Now let's get started developing.
